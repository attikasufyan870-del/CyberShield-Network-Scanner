# CyberShield: Network & Port Scanner

A lightweight, efficient Python-based cybersecurity tool designed to scan a target IP address or hostname for open standard ports. This tool helps identify potential security risks.

## Features
* **Hostname Resolution:** Converts domain names into IP addresses.
* **Smart Scanning:** Scans critical standard ports (SSH, HTTP, HTTPS).
* **Risk Identification:** Flags open ports that might expose vulnerabilities.
